# 一个用于内网互传文件和消息的工具
- node 22.20.0

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```
node build.js -v 1.0.0
```
[后端](https://github.com/mask47631/MaskServer)

## 使用
1. 运行后端服务
2. 点击页面右上角齿轮
3. 添加服务器地址写入后端服务地址
